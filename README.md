Este repositório é o meu laboratório pessoal de estudos. Aqui documento minha evolução na linguagem **Python**, explorando desde os fundamentos da programação até a construção de modelos de **Inteligência Artificial** e análise de dados.

---

## 🎯 Objetivos do Repositório
* Dominar a sintaxe e as boas práticas (PEP 8).
* Manipular grandes volumes de dados de forma eficiente.
* Criar visualizações de dados que contam histórias.
* Implementar modelos de Machine Learning e Deep Learning.

## 🛠️ Tecnologias e Bibliotecas Principais

### 📊 Data Science & Análise
- **Pandas**: Manipulação e análise de dados tabulares.
- **NumPy**: Computação numérica e matrizes multidimensionais.
- **Matplotlib/Seaborn**: Visualização de dados e gráficos estatísticos.

### 🤖 Inteligência Artificial
- **Scikit-learn**: Algoritmos clássicos de Machine Learning (Regressão, Classificação, Clustering).
- **TensorFlow / PyTorch**: Construção de Redes Neurais e Deep Learning.
- **NLTK / Spacy**: Processamento de Linguagem Natural (NLP).
